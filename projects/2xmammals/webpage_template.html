Genome-wide survey of sitewise selective pressures in mammals

The EnsEMBL Compara team and Goldman Group, EMBL-EBI, Hinxton UK
Gregory Jordan, Albert Vilella, Tim Massingham, Ewan Birney and Nick Goldman

The datasets and UCSC genome browser tracks below contain the results from calculating the site-wise selective pressures acting on protein-coding regions of all available vertebrate genomes, including the 22 mammals sequenced as a part of the Mammalian Genome Project.

Evolutionary pressures in four subtrees were considered in this analysis: vertebrates, primates, glires, and laurasiatheria. All of the datasets and many of the browser tracks below contain separate tracks from independent calculations on each subtree, allowing for direct comparison of evolutionary pressures between distinct mammalian subtrees at various scales.

We used the protein-based gene tree alignments from the Ensembl Compara database [1] and the Sitewise Likelihood Ratio method [2] to calculate dN/dS estimates at every protein-coding codon. Various filtering steps were employed in order to minimize errors resulting from alignment, sequencing and annotation error.

UCSC Tracks

  • Sitewise likelihood statistic: SITES_FILES

    A signed version of the SLR test statistic was used to generate a UCSC track, provided here in bigWig format. A value below 0 indicates evidence for negative selection, while a value above 0 indicates evidence for positive selection. The larger the magnitude of the value, the stronger the evidence for positive or negative selection.

Raw Data

  • Tabular results from the analysis are available here: Mammals SLR Tables

    This compressed data file can be opened using R. A number of objects will become available upon loading the file using R's load() function:

    • sites_[m|p|g|l]: data files corresponding to SLR estimates in four different clades: mammals, primates, glires, and laurasiatheria.
    • sites_info: information on the fields contained in the sites_[m|p|g|l] files.
    • genes_[m|p|g|l]: data files corresponding to one row per alignment from which the sites_[m|p|g|l] data were derived.
    • genes_info: information on the fields contained in the genes_[m|p|g|l] files.
  • All data available from this webpage (including gene alignments, trees, SLR results, and plots) can also be downloaded here (4gb .tar.gz): Mammals SLR Dataset
